# 🎯 Die beste gemeinsame Anwendung: **Anomaly / Outlier Detection**

Das ist mMn der **Gold Standard für deinen Vergleich**.

Warum?

## ✅ Alle Methoden liefern genau das, was du brauchst

* KDE → explizite Dichte ( \hat f(x) )
* log-concave MLE → explizite Dichte ( \hat f(x) )
* SM → Score ( \nabla \log \hat f(x) ), daraus:

  * Energie / relative Dichte
  * Gradientenfluss

👉 Alle drei liefern eine **„typicality measure“**

---

## ✅ Klassischer Zusammenhang (passt zu deinem Text!)

Du hast ja schon:

> LOF, Outlier Detection, Density-based methods

Perfekt.

Denn:

👉 Outlier Detection basiert genau auf:
[
\text{low density} \Rightarrow \text{outlier}
]

Das ist exakt die Brücke.

---

# 🧠 Der Trick: gleiche Entscheidung statt gleiche Dichte

Du brauchst **keine gleiche Dichte**
Du brauchst nur:

> eine Funktion, die Punkte ranked: „typisch vs untypisch“

---

# 🔧 Konkret: Wie du vergleichst

## Für KDE / MLE:

```r
score(x) = log f_hat(x)
```

## Für SM:

Du hast keinen log f, aber:

👉 Verwende:

### Option A (einfach & sehr stabil)

```r
score(x) = -|| s_hat(x) ||^2
```

Interpretation:

* große Gradienten → steile Regionen → oft Rand / low density
* kleine Gradienten → nahe Mode → high density

👉 funktioniert überraschend gut

---

### Option B (besser, wenn möglich)

```r
score(x) = - divergence(s_hat(x))
```

= approximierte Laplace von log f
